# Subtitle Renamer 

This is a **bulk renaming script** designed to organize `.srt` subtitle files downloaded from [Downsub.com](https://downsub.com). 

**Might update this in future with proper regex, but for now it's aimed for downsub only.**

It lets you automatically group and rename subtitles by **language**, **episode number**, or **both**.
